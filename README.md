### Usage

| Use                    | Command            | URL                                            |
| ---------------------- | ------------------ | ---------------------------------------------- |

| Start local dev server | `npm dev`          | [http://localhost:5173](http://localhost:5173) |
| Static build           | `npm build`        |                                                |
| View component library | `npm storybook`    | [http://localhost:6006](http://localhost:6006) |
